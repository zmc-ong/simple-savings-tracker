## Simple Savings Tracker

### 📝 Description
A web-based personal finance application. The Simple Savings Tracker helps users monitor their savings goals and daily expenditures through an intuitive interface with visual progress tracking. Built with vanilla HTML, CSS, and JavaScript on the frontend, with PHP and MySQL handling backend operations and secure authentication.

### 🚀 Features
- 🔐 **Secure User Authentication** - User registration with email validation, password hasing and session management
- 💰 **Savings Goal Management** - Create savings goals with target amount and deadlines, update progress in real-time, progress bars showing completion percentage
- 📝 **Daily Expense Tracking** - Expense logging with customizable categories, date-based entry system, view and edit past entries
- 📊 **Visual Dashboard** - Progress bars, summary cards, recent transactions, goal status
- 📈 **Monthly Financial Overviews** - Total income vs. expenses breakdown, progress toward monthly savings targets, data comparison

### 🛠️ Technologies
- HTML5
- CSS3
- JavaScript
- PHP 8.0+
- MySQL 5.7+
